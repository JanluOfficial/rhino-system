# rhinosystem

Upgrade and system info tool for Rhino Linux

# Building
Following dependencies are required
- gtk4
- libadwaita
- py-cpuinfo
- meson
- ninja

To build and install rhino-system simply run
```
meson setup _build
ninja -C _build
```
